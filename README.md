# relewise-sdk-java
Java SDK for Relewise

## Installation

Install via Maven.

```XML
<dependency>
    <groupId>com.relewise.client</groupId>
    <artifactId>relewise-sdk</artifactId>
    <version>1.0.0-alpha.2</version>
</dependency>
```
